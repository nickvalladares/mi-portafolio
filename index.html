    Luego de finalizar mis estudios, inicie mi proyecto de Tesis, un sistema hecho con el framework Laravel, en medio de este proceso me vi en la necesidad de trabajar, para lo cual me postule a diferentes ofertas laborales, es así como SmarTech se contacta conmigo y luego de una entrevista y posteriormente una prueba me brindan una pasantía de 3 meses y luego si cumplía con las expectativas un puesto laboral como desarrollador, así fue, estuve en la empresa por alrededor de 3 años, realizando tareas de diseño, creación, modificación y solucionando bugs del sistema de la empresa, también diseñaba y creaba sitios web con WordPress para los clientes que adquirían el sistema; entre las actividades más destacadas que hizo fueron integrar el servicio de Bucket de Google, la integración de la API de Telegram para proporcionar un servicio de notificaciones a los usuarios y la implementación de un logueo mediante OAuth2 para realizar calendarizaciones de actividades en Google Calendar desde el sistema de la empresa. Mientras trabajaba en SmarTech me llego la oportunidad de Trabajar en el Ministerio de Salud de El Salvador como desarrollador, luego de entrevista y prueba me quede con el puesto y actualmente estoy trabajando en diferentes sistemas del área de salud del país, estos sistemas están hechos con diferentes tecnologías como PHP puro, Symfony, Laravel, Node.js, Vue.js entre otras, por eso estoy familiarizado en diferentes tecnologías de desarrollo así como diferentes tipos de base datos como PostgreSQL, MySQL, MariaDB y MongoDB.
